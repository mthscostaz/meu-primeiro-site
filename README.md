
# Meu Primeiro Site

Este é meu primeiro projeto de site publicado no GitHub Pages.

O objetivo deste projeto foi praticar a estrutura básica de uma página web usando HTML e CSS, entendendo como organizar um projeto, criar arquivos, fazer commits e publicar um site online.

## 🔗 Site online

Acesse o projeto publicado:

https://mthscostaz.github.io/meu-primeiro-site/

## 🚀 Tecnologias usadas

- HTML
- CSS
- GitHub
- GitHub Pages

## 📚 O que aprendi

Neste projeto, aprendi os primeiros fundamentos de criação de sites:

- Como criar um repositório no GitHub
- Como criar um arquivo `index.html`
- Como conectar o HTML com o CSS
- Como criar um arquivo `style.css`
- Como fazer commits
- Como publicar um projeto usando GitHub Pages
- Como organizar um projeto simples de portfólio

## 🎯 Objetivo do projeto

Criar uma primeira página pessoal simples, visualmente organizada e publicada online, servindo como ponto inicial da minha evolução em desenvolvimento web.

## 🧠 Próximas melhorias

- Melhorar a responsividade
- Adicionar uma seção de projetos
- Adicionar animações simples com CSS
- Criar uma versão com JavaScript
- Melhorar a acessibilidade da página
